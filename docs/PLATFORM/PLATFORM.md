# PURPOSE

The Platform Layer defines how ORBIS operates after the Foundation.

The Foundation defines the permanent principles of the system.

The Platform Layer defines the shared runtime architecture used by every
module.

Every Business Module depends on this Platform.

## PLATFORM SCOPE

The Platform Layer includes:

• Landing Page

• User Registration

• Login

• Identity

• User Dashboard

• Workspace

• Module Launcher

• Admin Login

• Admin Dashboard

• Module Management

==============================================================================

## NOT INCLUDED

The Platform Layer does NOT contain business logic.

Business Modules such as:

• Farmer Brain

• DigiLedger

• AI Orchestrator

• Dairy

• Retail

will be implemented separately.
